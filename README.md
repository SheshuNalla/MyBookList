# My Book List App

- An application built by "html", "javascript" and no frameworks involved.
- Styling is done by "tailwind css".
- The "local storage" is used as the Database to store and retrieve the data

# About Application

- Application to store the list of book collections.
- We can add the books and delete the books dynamically.
- Given the functionality to save and delete the data directly from the local storage.
- Given the functionality to edit and update the existing things.
- Given Search functionality to search the specific book when we needed

# Application contains

* -Header
    - Logo
    - Heading
* -Body
    - Book form to  validate the book
    - "Add book" button
    - The table for the listed book items
    - Edit button to edit the existing content and update it
    - Search functionality to search the specific book when we needed.
